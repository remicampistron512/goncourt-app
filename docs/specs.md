### Pour l’utilisateur, l’application permet :

#### Consultation des sélections

* afficher la liste des livres d’une sélection :

  * choisir la sélection : 1ère, 2e, 3e ou finale
  * voir pour chaque livre : titre, auteur, éditeur

* consulter le détail d’un livre :

  * voir toutes les informations du livre :

    * titre, résumé, auteur, éditeur
    * date de parution, nombre de pages, ISBN, prix éditeur
    * principaux personnages

---

### Pour le président du jury, l’application permet :

#### Gestion des sélections

* définir la 2e sélection :

  * afficher les livres de la 1ère sélection
  * choisir les livres retenus pour la 2e sélection (jusqu’à 8 livres)

* définir la 3e sélection (finalistes) :

  * afficher les livres de la 2e sélection
  * choisir les livres finalistes pour la 3e sélection (jusqu’à 4 livres)

#### Gestion du dernier tour de scrutin

* saisir les voix du dernier tour :

  * afficher les livres finalistes
  * saisir le nombre de voix obtenu par chaque livre

* consulter les résultats du dernier tour :

  * afficher le lauréat (livre ayant le plus de voix)

---

### Pour un membre du jury (version future, optionnelle) :

* s’authentifier :

  * saisir ses identifiants pour accéder à son espace jury

* voter :

  * voter pour les livres de la 2e sélection
  * voter pour les livres de la 3e sélection
  * voter pour le lauréat parmi les finalistes
